# ⌨️ Typing Pro

<br/>

### 📑 프로젝트 설명

- 한글 및 영어로 타자 연습을 할 수 있는 웹 플랫폼 입니다.
- 카테고리에서 원하는 글 또는 주제를 선택하여 타자 연습을 할 수 있습니다.

<br />

### 🔎 주요 기능

- 사용자가 프로필 이미지와 닉네임 입력
- 카테고리에서 작성할 언어와 글을 선택
- 긴 글, 단어, 게임 모드로 구성
- 타자 연습 진행 시간, 실시간 속도, 정확도 확인
- 입력하는 글자 색으로 쉬운 오타 확인
- localStorage에 저장된 사용자의 타자 연습 기록과 통계 확인
